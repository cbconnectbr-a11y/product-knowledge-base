# 系统更新日志 - 2026-05-17

## 重大更新

### 1. ✅ 对话上下文与追问功能

**功能描述：**
智能识别用户追问，自动补充上次对话中的SKU，无需重复输入。

**实现内容：**
- ✅ 会话管理（记忆最近3轮对话，30分钟过期）
- ✅ 追问识别（强/弱关键词分类）
- ✅ 查询增强（自动补充上次SKU）
- ✅ 多语言支持（中文/葡语自动切换）
- ✅ RAG上下文集成

**使用示例：**
```
用户: "ANKI-001 产品尺寸多少"
AI: "长22cm、宽19cm、高7.8cm"

用户: "价格是多少"  ← 自动理解为ANKI-001的价格
AI: [基于上下文回答ANKI-001的价格]

用户: "说明书在哪里"  ← 继续追问ANKI-001
AI: [返回ANKI-001的说明书链接]
```

**技术实现：**
- `bot/session_manager.py` - 会话管理核心模块
- `bot/handlers.py` - 集成追问检测和查询增强
- `bot/rag.py` - 支持对话上下文的RAG生成
- `scripts/utils.py` - 语言检测工具
